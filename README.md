## Getting Started  

### Prerequisites  
Ensure you have **Node.js** and **npm** installed.  

### Installation  
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/whispr.git
   cd whispr
   ```
2. Install dependencies for both frontend and backend:  
   ```bash
   cd client
   npm install
   cd ../server
   npm install
   ```
3. Start the development servers:  
   - **Frontend:**  
     ```bash
     cd client
     npm start
     ```
   - **Backend:**  
     ```bash
     cd server
     npm start
     ```
